First backbone model

Create person with default values

> var person = new Person()
  undefined

Check data in the model

> person.age
  undefined
> person.first_name
  undefined
> person.last_name
  undefinde

We use the method 'get'

> person.get('first_name')
  Anton
> person.get('last_name')
  Larkin
> person.get('age')
  19

Use method 'work'

> var person = new Person()
  undefined
> person.work()
  "Anton Larkin is working."

Create person with user values

> var person = new Person({first_name: 'Jeffrey', last_name: 'Way', age: 27})
  undefined
> person.toJSON()
  Object

Method 'set'

> person.get('occupation')
  "worker"
> person.set('occupation', 'Django Developer')
  d
> person.toJSON()
  Object
